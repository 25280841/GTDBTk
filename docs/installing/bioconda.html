Bioconda¶

Installation¶

  1. Create the conda environment:

    • Latest version: conda create -n gtdbtk -c bioconda gtdbtk

    • Specific version: conda create -n gtdbtk-1.1.0 -c bioconda gtdbtk=1.1.0

  2. Activate the environment: conda activate gtdbtk

Post-install¶

  1. Download the reference package either manually or by running download-db.sh.

  2. Set the GTDBTK_DATA_PATH environment variable in {gtdbtk environment path}/etc/conda/activate.d/gtdbtk.sh to the reference package location.
